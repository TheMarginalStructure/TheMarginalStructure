# The Marginal Structure — 边际结构

> **"测绘黑暗，设立路标，而非征服。"**
>
> 一个秘密国际组织，致力于勘探、记录和隔离那些存在于现实边缘的异常空间——"阈界"。

---

## 世界观

现实并非坚固稳定。在人类集体意识、物理定律和未知维度相互作用的边缘地带，存在着被称为 **阈界（Threshold）** 的渗透层或断层空间。它们并非另一个世界，而是现实结构中的 bug、褶皱或溢出地带。

边际结构（The Marginal Structure）是一个力量有限的秘密组织。他们自称"勘探者、记录者、隔离者"，而非"收容者"。目标是防止阈界现象对大众现实造成灾难性影响，并艰难地从中获取知识和资源。

### 阈界类型

| 类型 | 描述 |
|------|------|
| 认知阈界 | 由集体或个体意识塑造，进入者的恐惧会滋养并改变它 |
| 物理/数学阈界 | 物理常数异常、数学规律扭曲的非线性空间 |
| 历史/时间阈界 | 时间流碎片化，未被选择的历史时间线 |
| 信息阈界 | 纯粹由信息构成的领域，可能导致认知被同化 |
| 混合/未知阈界 | 无法归类，完全超出理解 |

### 组织架构

| 部门 | 职责 |
|------|------|
| 外勤行动部 | 勘探者团队、守护卫者部队、联络与掩盖部门 |
| 档案与研究部 | 档案员、分析员、实验室团队 |
| 医疗与心理部 | 生理医疗、心理评估与康复、净化协议 |
| 安全与防护部 | 安全专员、防护技术、应急响应 |
| 后勤与架构部 | 架构师、后勤支援、"锁匠"连接点技术组 |

### 威胁等级系统

**颜色代码**：白（情报不足）→ 蓝（无害）→ 绿（可利用）→ 黄（不稳定）→ 琥珀（主动危险）→ 红（存在性威胁）→ 黑（文明终结级）

**字母代码**：E-实体 / P-现象 / C-认知/模因 / T-时间 / I-信息 / O-存在性

> 示例：**"永夜钟楼" V-990-INC** 评为 **琥珀色-C**（主动危险，认知危害）

## 仓库结构

```
TheMarginalStructure/         ← 父工作区（此仓库）
├── content/                   ← 世界观设定、档案、系列等文本内容
│                              → git@tms:TheMarginalStructure/ThresholdArchive.git
├── frontend/                  ← 前端应用 (React + Vite + Tailwind)
│                              → git@tms:TheMarginalStructure/ThresholdArchive-Frontend.git
├── backend/                   ← 后端 API (Express + Prisma + TypeScript)
│                              → git@tms:TheMarginalStructure/ThresholdArchive-Backend.git
├── start.ps1                  ← 一键启动脚本
└── README.md
```

## 快速开始

```powershell
# 克隆（含子模块）
git clone --recurse-submodules git@tms:TheMarginalStructure/ThresholdArchive.git TheMarginalStructure

# 如果已克隆，初始化子模块
git submodule update --init --recursive

# 启动前后端服务
./start.ps1
```

## 子模块更新

```powershell
# 拉取所有子模块最新代码
git submodule update --remote --merge

# 或逐个仓库操作
cd content && git pull
cd ../frontend && git pull
cd ../backend && git pull
```
